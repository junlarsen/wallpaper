# Wallpaper

This is the wallpaper with live audio visualizers I use for Wallpaper Engine.

![Background](assets/showcase.png)

## Requirements

- Wallpaper Engine (Steam)
- NodeJS
- NPM

## Setup

- Clone the repo
- Run `npm install`
- Run `npx webpack`
- Open Wallpaper Engine
- Set the background to the index.html file
