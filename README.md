# Wallpaper

This is the wallpaper I use for Wallpaper Engine.

![](assets/showcase.png)

It also has live audio visualizers

## Requirements

- Wallpaper Engine (Steam)
- NodeJS
- NPM

## Setup

- Clone the repo
- Run `npm install`
- Run `npx webpack`
- Open Wallpaper Engine
- Set the background to the index.html file
